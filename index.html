Spider

A string manipulation language




Ranges are inclusive. Also, if a range's end is not given, the range will end at the end of the string.

Char Arguments Description
¢ None In a replace command, represents the character being replaced.
o None Reverses the string
d search:<string> Deletes all occurrences of the argument
D search:<string>, start:<int>, end:[int] Deletes all occurrences of the first argument in the given range
i start:<int>, end:[int] Deletes all characters in the given range
I start:<int>, end:[int], replacement:[string] Replaces a given index range with the third argument
r search:<string>, replacement:<string> Replaces all occurrences of the first argument with the second
R search:<string>, replacement:<string>, start:<int>, end:[int] Replaces all occurrences of the first argument in the given range with the second
m search:<string> Removes all occurrences of the given string except for the first
s replacement:<string> finds:<string[]> Sequential replace - First find is found and replaced, second find is found after first and replaced, etc. If finds is one string, it will be split up into multiple characters as the finds
